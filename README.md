# Tiny Language Model

A small transformer-style language model built from scratch in PyTorch.

## What it is
A language model that learns to predict the next word in a sentence. 
Built to understand how large language models like GPT work under the hood.

## How it works
The model follows the transformer architecture:
- **Tokenization** — words are converted to IDs
- **Embeddings** — IDs become vectors that capture meaning
- **Attention** — words learn to relate to each other
- **Generation** — the model predicts the next word one at a time

## How to run it
pip install torch numpy
python language_model.py
